Operate z/OS Jobs
=================

**IBM z/OSMF collection** provides capabilities to work with batch jobs on a z/OS system using Ansible playbooks. There are three roles provided to interact with z/OS jobs REST services.

Roles
-----

* `zmf_job_complete`_:

   This role can be used to submit a job to run on z/OS, and check its return code after the job is completed.
